# MYChess

A web based chess app with pass & play mode and personality driven AI opponents built with Flask and Stockfish 18.

🔗 **[Play Now](https://chess-website-zxwb.onrender.com)**

---

## Features

- Play against four AI bots, each with unique difficulty and personality
- Pass & Play mode for two players on the same device
- Bots react to your moves with live dialogue — blunders, checks, captures, and more
- Save completed games and replay them move by move
- User accounts with game history

---

## Bots

| Bot | Elo | Style |
|-----|-----|-------|
| Rookie | ~400 | Just learning the game. |
| Balanced | ~1200 | Solid and hard to trick. |
| Aggressive | ~1800 | Attacks at every opportunity. |
| Grandmaster | ~2400 | Ice cold, sees everything. |

---

## Built With

- **Flask** — backend and routing
- **chess-api.com** — Stockfish 18 engine (no local install needed)
- **chessboard.js + chess.js** — board rendering and move validation
- **SQLite** — game storage
- **Render** — hosting
